## 기술 스택

- **프레임워크**: Next.js 14 (App Router)
- **언어**: TypeScript
- **스타일링**: Tailwind CSS

## 설치 및 실행 방법

```
// # 저장소 클론
git clone [repository-url]
npx create-next-app@14
// # 라이브러리 설치
npm install @heroicons/react
npm i zod
npm i validator
npm i @types/validator
```

## Setting

```
npm i zod
// 최신 버전 (6.6.0 - 250424 기준 )
```
